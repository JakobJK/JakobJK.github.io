mx

mx is a node application cleverly disguised as a CLI Tool used to resize images directly in the terminal.
I have set up an alias for mx as follows:

alias mx='node /Users/jake/mx/index.js'

This will now allow you to resize any image by entering 'mx' followed by a relative path to the image and an option parameter. An example would look like

mx ./test.jpg w200

Which will then output a new image, which is a copy of test, but have the size of 200 pixels wide (height remains in proportion). Feel free to check out mx on GitHub
