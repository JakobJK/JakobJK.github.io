modelChecker

modelChecker is a tool written for Autodesk Maya to sanity check digital polygon models for production. It aims to be as unopinionated as possible, It gives you concise feedback, and let's you select your error nodes easily. The modelChecker is writtin in Python (Maya API), PySide2 (pyQt).

The modelChecker can be downloaded on Gumroad

The modelChecker is open source and available on GitHub here
