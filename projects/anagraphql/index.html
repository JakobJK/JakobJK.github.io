AnagraphQL

AnagraphQL is an NPM package, express middleware, which sits before your GraphQL end point. It allows you to write thorough policies on how your GraphQL end point is allowed to be quired. It comes with a fully server side rendered playground for you to test out your policies. AnagraphQL was built with the purpose of joining the conversation of how to safeguard public facing GraphQL API's.

You can check out anagraphql at NPM and GitHub

AnagraphQL was built by Alfredo Alpizar, Camera Kendall, Jackie Chow, and my self.
